# ImageProcessing
Цифровая обработка изображений
1. Средствами библиотеки numpy создать массив со случайными значениями, распределенными по нормальному закону. В массиве максимальный элемент заменить на -1000, минимальный на 1000.
2. Эквализация интенсивности изображения.
3. Изучение метрик качества изображений, линейная фильтрация в пространственной и частотной областях. 
4. Бинаризация.
